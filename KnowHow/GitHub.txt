Fazer update do repositorio local:
git pull

criar branch:
git checkout -b nome

Adicionar arquivos:
git add . -A

Fazer commit:
git commit -m "Changes in the schematic and component list."

remover ultimo commit local:
git reset --soft HEAD^

fazer merge do branch criado com o master:
git checkout master
git merge nome
(caso houverem conflitos basta editar o arquivo em conflito, adiciona-lo novamente e fazer o commit j√° no branch master)

deletar branch
git branch -d nome

enviar commits
git push
